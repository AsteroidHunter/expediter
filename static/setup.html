Connecting…

Checking the secure connection to your Mac.

One-time setup

To connect securely (and to use the microphone for voice), your phone needs to trust this Mac's certificate. You only do this once per phone.

Download certificate

  1. Tap Download certificate, then tap Allow.
  2. Open Settings → General → VPN & Device Management, tap the Expediter Local CA profile, and Install it.
  3. Open Settings → General → About → Certificate Trust Settings and turn on full trust for Expediter Local CA.
  4. Come back to this page — it connects automatically.

This must be done in Safari — other browsers can't install the profile. Prefer to skip it? Run expediter --http on your Mac for a plain connection (no microphone).
